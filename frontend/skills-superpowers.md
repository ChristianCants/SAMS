# obra/superpowers (14 skills)
Source: https://github.com/obra/superpowers

## Planning & Execution
- **brainstorming** - Transform rough ideas into fully-formed designs through structured questioning and alternative exploration.
- **writing-plans** - Create structured plans before executing complex tasks to ensure alignment and catch issues early.
- **executing-plans** - Execute pre-written plans step by step with checkpoints and verification.
- **using-superpowers** - Meta-skill for understanding and getting the most out of the superpowers skill collection.

## Development Workflow
- **test-driven-development** - Use when implementing any feature or bugfix, before writing implementation code. Write tests first.
- **subagent-driven-development** - Dispatches independent subagents for individual tasks with code review checkpoints between iterations.
- **finishing-a-development-branch** - Guides completion of development work by presenting clear options and handling chosen workflow.
- **using-git-worktrees** - Creates isolated git worktrees with smart directory selection and safety verification.

## Debugging & Quality
- **systematic-debugging** - Structured approach to diagnosing and fixing bugs methodically rather than guessing.
- **verification-before-completion** - Verify work meets requirements before marking tasks as done; catches issues early.

## Code Review
- **requesting-code-review** - How to properly request a code review with the right context and framing.
- **receiving-code-review** - How to receive, interpret, and act on code review feedback constructively.

## Writing
- **writing-skills** - Guidelines and techniques for producing clear, well-structured written content.

## Parallel Work
- **dispatching-parallel-agents** - Spawn and coordinate multiple independent agents to work on tasks simultaneously.
